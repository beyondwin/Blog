# Curated Notes

`docs/notes/`는 사람이 다듬은 장기 보관 문서를 둔다. source capture와 generated navigation이 아니라, 나중에 직접 읽고 인용할 수 있는 library layer다.

운영 규칙:

- 안정적인 topic 아래에 파일을 둔다.
- topic을 너무 깊게 쪼개지 않는다. retrieval에 도움이 될 때만 새 folder를 만든다.
- 문서를 추가, 이동, rename, 크게 수정하면 `docs/_index/catalog.yml`을 갱신한다.
- 새 topic이 장기 retrieval category라면 `docs/_index/topics.yml`도 갱신한다.
- generated `docs/wiki/`와 `graphify-out/`은 이 폴더의 대체물이 아니다.
